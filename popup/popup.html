This extension uses the Giant Bomb API. This API requires the use of an API key, and the extension has a default key. However, the API has usage rate limits per API key, so it is recommended that you provide your own for better reliability. You can find your key by logging in to Giant Bomb and navigating to giantbomb.com/api. Your key will never be logged or shared by Giant Bomb QoL.
